Safety Video Harness Plugin

교육자료를 넣으면 주제 추출, 스토리보드, 스타일 잠금, Codex imagegen 키프레임, QA/RALPH, Seedance 계획까지 이어지는 산업안전 영상 제작 플러그인 구조다. 유료 영상 생성은 Gate 2 이후에만 실행한다.

7에이전트 패키지
6로컬 스킬
13내장 reference 요약
71현재 테스트 통과 수

실행 흐름

01

인터뷰

자료, 주제, 길이, 이미지 분량, 레퍼런스, 스타일, 승인 범위를 묻는다.

02

자료 추출

PPTX/SOP를 등록하고 렌더링한 뒤 주제 후보와 source facts를 만든다.

03

스토리보드

교육자료 근거로 장면, 자막, start/end keyframe 계약을 작성한다.

04

스타일 잠금

첫 장면과 style guide로 인물, 차량, 공간, 카메라 bible을 고정한다.

05

프롬프트 팀

장면별 프롬프트 초안은 병렬화하고 최종 imagegen은 중앙 통제한다.

06

이미지 생성

Codex 내장 imagegen으로 프로젝트 draft 이미지를 만들고 버전을 보존한다.

07

QA/RALPH

점수, blocker, 반복 이력, llm-wiki memory로 기준 미달 이미지를 개선한다.

08

영상 계획

승인 이미지로 Seedance dry-run과 비용 산정 후 Gate 2에서 live 여부를 결정한다.

가드레일

Live 차단

Gate 승인과 외부 업로드 허용 전에는 live imagegen/Seedance/TTS를 차단한다.

No Narration

현재 범위는 영상+자막/오버레이. TTS와 나레이션은 후속 과제다.

Paid Guard

영상 생성은 비용 고지와 Gate 2 승인 후에만 진행한다.

Version Preserve

draft/approved 이미지와 산출물은 덮어쓰지 않고 버전을 보존한다.

Evidence

테스트, dry-run, QA 결과는 evidence/와 qa/에 남긴다.

RALPH

이미지 루프는 최대 20회 early stop, 같은 blocker 3회면 upstream 수정으로 전환한다.

레퍼런스

OpenAI Image

이미지 생성, 참조 이미지, 일관성 한계, 텍스트 렌더링 한계.

OpenAI Prompt

구조화 지시, 복잡한 작업 분해, 참조 텍스트 제공.

OpenAI Video

이미지 레퍼런스, 캐릭터 재사용, 영상 연속성 개념.

Higgsfield CLI

Seedance/Higgsfield 인증, 생성 명령, 크레딧 정책.

MCP

플러그인형 도구, 리소스, 프롬프트 연결 구조.

FFmpeg / JSON Schema

영상 검사, 메타데이터, 프레임 추출, 계약 검증.
