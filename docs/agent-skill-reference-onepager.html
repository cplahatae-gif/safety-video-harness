제작 파이프라인에서의 위치

topic-extractor

교육자료에서 여러 안전 주제 후보를 추출하고 출처를 유지한다.

story-writer

선택 주제를 30초 교육 스토리보드로 바꾼다. 나레이션은 제외한다.

lead-style-agent

첫 키프레임을 기준으로 스타일·인물·차량·공간 bible을 잠근다.

scene-prompt-agent

장면별 이미지 프롬프트 초안을 병렬 작성한다.

visual-director-arbiter

프롬프트 세트를 중앙 통합하고 실제 imagegen 실행 가능 여부를 결정한다.

image-consistency-check

생성 이미지를 스토리보드·스타일·연속성 기준으로 평가한다.
