정밀 산업 웹툰 키프레임 제작 맵

레미콘 사업장 충돌 예방 교육 영상을 위해 스토리보드 기준 7장 키프레임을 만든 현재 흐름이다. 첫 장은 리드 스타일 기준으로 재사용하고, 나머지 장면은 제작팀 preflight가 프롬프트 일관성을 잠근 뒤 한 coordinator가 순차 생성한다.

7총 키프레임 수
6이번에 새로 생성한 장면
30s목표 영상 길이
v002최신 로컬 이미지 세트

스토리 흐름

sc01

동선 분리

BCT와 덤프트럭, 보행자 동선을 먼저 분리해서 위험 맥락을 세운다.

sc02

사각지대 통제

후진 경로 옆 위험구역을 비우고 작업자를 보행로 밖으로 나가지 않게 한다.

sc03

신호수 정지 신호

신호봉과 무전기로 운전자에게 정지·대기 신호를 명확히 전달한다.

sc04

운전자 확인

후방카메라, 거울, 신호수를 확인한 뒤 저속 후진을 준비한다.

sc05

차량 통로 진입 차단

작업자가 통로로 들어가려는 순간 감독자가 안전 동선으로 돌려보낸다.

sc06

보행·속도 수칙

아이콘 표지판과 보행로로 속도·스마트폰 금지·PPE 수칙을 보여준다.

sc07

안전 종료

차량, 보행자, 신호수가 모두 통제된 상태로 예방 스토리를 마무리한다.

최신 로컬 키프레임 세트

생성 규칙

스타일 고정

korean-industrial-webtoon 스타일 가이드를 기준으로 선화, 셀 셰이딩, 성인 작업자 비율, 산업 장비 형태를 유지한다.

연속성 고정

같은 BCT, 같은 덤프트럭, 같은 보행로, 같은 위험구역, 같은 PPE 색상을 모든 장면에 반복한다.

교육성 우선

인물의 시선은 보이는 위험, 신호수, 운전자, 거울, 카메라, 안전 동선 중 하나를 향해야 한다.

에이전트 앞단

Lead Style Agent

agents/lead-style-agent/AGENT.md가 첫 키프레임을 기준으로 style, character, vehicle, space, camera, rendering bible을 만든다.

Scene Prompt Agents

agents/scene-prompt-agent/AGENT.md가 장면별 프롬프트 초안을 병렬 작성하되, 직접 이미지를 생성하지 않는다.

Visual Director Arbiter

agents/visual-director-arbiter/AGENT.md가 전체 프롬프트 세트를 통합 검토하고 중앙 imagegen 실행 가능 여부를 결정한다.
