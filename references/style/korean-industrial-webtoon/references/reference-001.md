# Reference 001

Codex imagegen으로 생성한 레미콘 사업장 충돌 예방 교육용 웹툰풍 기준 이미지다.

## 참고할 것

- 현대 한국 웹툰풍 선화와 셀 셰이딩
- 안전모, 형광 조끼, 차량, 보행자 동선을 명확히 읽히게 하는 색상 대비
- 산업현장을 단순화하면서도 레미콘 플랜트, BCT, 덤프트럭 같은 핵심 요소를 유지하는 방식
- 교육 장면으로 사용할 수 있는 밝고 통제된 분위기

## 복제하지 말 것

- 특정 회사 로고나 현장 식별 정보
- 이미지 안의 임의 텍스트
- 컷별 인물 수나 위치를 그대로 고정하는 것
- 스토리보드와 맞지 않는 시선 방향이나 갑작스러운 인물 출현/소실
